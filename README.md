# linux-peripheral-drivers-c
C Hardware Abstraction Layer (HAL) for Linux-based SBCs (Orange Pi, Raspberry Pi). Includes high-performance drivers for I2C, SPI, UART, PWM, GPIO.


> ### ⚠️ ADVERTENCIA DE RENDIMIENTO
> **Aquí no hay 'import' mágicos ni scripts lentos.** Este es un entorno exclusivo para desarrolladores de **C**. 
> 
> Si buscas la comodidad de Python sobre el rendimiento del "metal", te sugerimos buscar otros repositorios. Aquí programamos para el hardware, no para el intérprete. Priorizamos el control total de la memoria y la eficiencia del ciclo de reloj.
